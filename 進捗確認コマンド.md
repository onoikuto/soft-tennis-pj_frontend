# Flutter実行の進捗確認コマンド

## 1. Flutterプロセスの確認

Flutterが実行中かどうかを確認：
```bash
ps aux | grep flutter
```

実行中のFlutterプロセスが表示されます。

## 2. ネットワーク接続の確認

Flutterがダウンロード中かどうかを確認：
```bash
lsof -i | grep flutter
```
または
```bash
netstat -an | grep ESTABLISHED | grep -E "(googleapis|storage)"
```

## 3. ディスクI/Oの確認

Flutterがファイルを書き込んでいるか確認：
```bash
sudo fs_usage -w -f filesys | grep flutter
```

## 4. ログファイルの確認

Flutterのログを確認（存在する場合）：
```bash
tail -f ~/.flutter/logs/*.log 2>/dev/null || echo "ログファイルが見つかりません"
```

## 5. キャッシュディレクトリのサイズ確認

ダウンロードの進捗を確認：
```bash
cd /Users/ononono/soft-tennis-pj_frontend
du -sh flutter/bin/cache
ls -lh flutter/bin/cache/dart-sdk 2>/dev/null
```

## 6. 簡単な確認方法（推奨）

最も簡単な確認方法：
```bash
cd /Users/ononono/soft-tennis-pj_frontend
export PATH="$PATH:/Users/ononono/soft-tennis-pj_frontend/flutter/bin"

# Flutterのバージョンを確認（これで進捗がわかります）
flutter --version

# または、doctorコマンドで状態を確認
flutter doctor -v
```

## 7. プロセスを強制終了したい場合

もしフリーズしているように見える場合：
```bash
# Flutterプロセスを探す
ps aux | grep flutter | grep -v grep

# プロセスIDを確認して終了（例: PIDが12345の場合）
kill 12345

# 強制終了する場合
kill -9 12345
```

## 8. リアルタイムで進捗を確認

別のターミナルで以下を実行すると、リアルタイムで確認できます：
```bash
watch -n 1 'ps aux | grep flutter | grep -v grep'
```
