# Dart VMクラッシュエラーの対処方法

## 発生したエラー
```
../../runtime/vm/cpuinfo_macos.cc: 42: error: unreachable code
```

## 原因
- macOS ARM64（Apple Silicon）マシンで、x64版のDart SDKがダウンロードされていた可能性
- Flutterのキャッシュに問題があった

## 対処済み
✅ Flutterのキャッシュをクリアしました

## 次のステップ

### 1. 再度依存関係をインストール
```bash
cd /Users/ononono/soft-tennis-pj_frontend
export PATH="$PATH:/Users/ononono/soft-tennis-pj_frontend/flutter/bin"
flutter pub get
```

このコマンドで、正しいアーキテクチャ（ARM64）のDart SDKが自動的にダウンロードされます。

### 2. アプリを実行
```bash
flutter run -d macos
```

## それでもエラーが出る場合

### 方法1: Flutterを再インストール
```bash
cd /Users/ononono/soft-tennis-pj_frontend
rm -rf flutter
git clone https://github.com/flutter/flutter.git -b stable --depth 1
export PATH="$PATH:/Users/ononono/soft-tennis-pj_frontend/flutter/bin"
flutter doctor
```

### 方法2: Web版で実行（回避策）
macOS版で問題がある場合、Web版で実行できます：
```bash
flutter run -d chrome
```

Web版でもSQLiteは動作します（sqflite_web_ffiを使用）。
